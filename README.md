# Calculadora-de-Programador_Programmer-Calculator
Português: Calculadora de Decimal, Hexadecimal e Binário. Este programa irá dar 3 opções de cálculo para o usuário: de Decimal para Hexadecimal e Binário; de Binário para Decimal e Hexadecimal; e de Hexadecimal para Decimal e Binário. Após a escolha, o usuário deve digitar no formato informado. Caso o usuário não o faça, o programa irá informar o erro cometido.

English: Decimal, Hexadecimal and Binary Calculator. This program will give 3 calculation options to the user: from Decimal to Hexadecimal and Binary; from Binary to Decimal and Hexadecimal; and from Hexadecimal to Decimal and Binary. After choosing, the user must type in the informed format. If the user doesn't do it, the program will inform the mistake made.
